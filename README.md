# pterodactyl-tw  
## 關於
pterodactyl panel的繁體中文翻譯版本，修改自原版`v1.11.2`及部分來自中國版`v1.11.2.0`。  
目前此翻譯版本存在許多不確定性，建議不要在實際環境中使用。  
我們也歡迎您使用pull request來協助我們翻譯、改進！
